## Every command starts with 7 unless its specified to have no prefix - Commands: - uptime (still half-broken) **Shows bot uptime... with dog years?** - ping **Gets the response time for the bot to Discord in miliseconds.** - say **Lets you make the bot say what you want.** - mariomaker **Use as `7mariomaker 1234-0000-ABCD-5678` This command is particularly slow.** - webp **Converts PNG and JPG files from direct links into Google's WebP format.** - shutdown **Shuts down the bot (Owner only)** - setavatar **Sets the bot's avatar (Owenr only) (Very broken)** - setgame **Sets the bot's currently played game (Owner only)** - setname **Sets the bot's global username (Owner only)** - ram **Shows the memory usage of the host system (currently broken)** - cpu **Shows the CPU usage of the host system** - test (useless) **Ignore this** - zeldathon **Links to the twitch channel for Zeldathon.** - help **link to a horrible and outdated word-cloud that has no purpose (needs fixing)** - Meme commands: - heyguys - player - tone-analyzer - kappa - sponge - ok - osu-ea - ubw - illuminati - grand-dad - mountain - dead-meme - sakurako1, sakurako2 and sakurako3 - windows - nvidia - desu, not-desu and not-not-desu - idgaf - joels-xbox and joels-xbox-uncut - doyourbest - ayy (No prefix) - mcyoutubers - hilary - creator - lang - commands - iwatch - serverinfo - music - google (Not working -_-) - graphics - noclear - slowmode - 777 (No prefix) - levels - weeb - rank
